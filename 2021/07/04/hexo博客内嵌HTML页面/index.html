hexo博客内嵌html页面

我是一个前端菜鸟，最近开始尝试用hexo写博客。平时自己会写一些小demo，每次想温习一下都只能在本地打开，找文件的过程又复杂至极。我是一个懒人，就想着能不能把HTML嵌入到自己的博文中，既可以记录用到的技巧，也可以看见最终效果，还省去了找源文件的琐碎，一举三得。

那么如何实现呢？我们可以利用markdown支持html标签的特性，来嵌入html代码。

1. 利用<iframe>

利用<iframe>标签，直接在文中嵌入完整的html页面。

iframe创造了一个隔离环境，可以避免嵌入页面的样式被博客页面的样式污染（也可避免博客站样式被嵌入页面污染）。

2. 直接嵌入HTML代码段。

纯粹的 HTML 代码，加上 style 和 script 标签，直接写在 .md 里面。

但当一篇博客有多个demo，每个demo之间很有可能会产生样式冲突，且也有可能导致嵌入页面与博客原本页面之间的样式冲突。
